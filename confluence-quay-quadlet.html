Quay Container Registry — Quadlet Deployment

Self-hosted Project Quay container registry running as Podman Quadlet systemd units. TLS is terminated directly by Quay (no reverse proxy).

Architecture

All three services run in host network mode as Podman Quadlet containers managed by systemd.
